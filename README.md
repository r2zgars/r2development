# r2 development site

Bu proje Vite + React ile hazırlanmıştır.

## Kurulum

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel'e ücretsiz yükleme

1. Bu klasörü GitHub repo'suna yükle.
2. Vercel hesabı aç.
3. "Add New Project" > GitHub repo'nu seç.
4. Framework olarak Vite otomatik algılanır.
5. Deploy'e bas.
6. İstersen sonra domain bağla.
